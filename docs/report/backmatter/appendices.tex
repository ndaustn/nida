\section{Ek-4: Kullanım Kılavuzu Özeti}

\subsection{Sisteme Giriş}

\begin{enumerate}
  \item Web tarayıcınızda sisteme gidin
  \item E-posta ve şifre ile giriş yapın veya yeni hesap oluşturun
  \item Dashboard sayfasına yönlendirileceksiniz
\end{enumerate}

\subsection{Çizim Analizi}

\begin{enumerate}
  \item ``Yeni Analiz'' butonuna tıklayın
  \item HTP çiziminin fotoğrafını yükleyin
  \item Çocuğun yaşını girin
  \item Çizim türünü seçin (Ev / Ağaç / İnsan / Tam HTP)
  \item ``Analiz Et'' butonuna tıklayın
  \item Sonuçları inceleyin
\end{enumerate}

\subsection{Sonuçları Yorumlama}

\begin{itemize}
  \item \textbf{Yüksek Güven:} Güçlü akademik destek
  \item \textbf{Orta Güven:} Orta düzeyde destek
  \item \textbf{Düşük Güven:} Sınırlı destek, dikkatli yorumlayın
\end{itemize}

\begin{warningbox}
NIDA bir tanı aracı değildir. Profesyonel değerlendirmenin yerini almaz. Endişe verici bulgularda uzman yönlendirmesi yapınız.
\end{warningbox}
